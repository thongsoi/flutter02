How to setup backend
like normal go project

How to setup frontend
$ cd frontend
$ flutter create .
# Replace pubspec.yaml and lib/main.dart with provided codes by claude

How to run
use multiple terminals
Terminal 1
$ cd backend 
$ go run main.go

Terminal 2
$ cd frontend
$ flutter pub get
$ flutter pub upgrade
$ flutter run

How to stop
Terminal 1 = Ctl C
Terminal 2 = Close emulator device (web/android/ios)
